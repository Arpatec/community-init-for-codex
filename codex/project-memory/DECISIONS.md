# DECISIONS

## Decision log

### 2026-03-30 - Positionnement du repo

- Le repo est positionne comme un equivalent communautaire de `/init` pour Codex et autres LLM, enrichi d'une memoire projet persistante.
- Raison : cette promesse explique clairement la valeur du depot pour un visiteur GitHub.

### 2026-03-30 - Documentation de premiere publication

- La premiere mise en forme publique reste en francais.
- Raison : alignement avec la documentation existante et reduction du temps de mise en ligne.
- Suivi potentiel : ajouter une version anglaise ou bilingue si la cible s'elargit.

### 2026-03-30 - Licence open source et attribution

- Le repo utilise la licence `Apache-2.0` avec un fichier `NOTICE`.
- Attribution retenue : `Arpatec / doc-rezo.fr`.
- Raison : autoriser la reutilisation tout en conservant plus clairement la trace du projet d'origine.
